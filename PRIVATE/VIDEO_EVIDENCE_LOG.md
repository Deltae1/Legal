## ⚠️ PURPOSE OF THIS FILE

Catalog of all video footage captured via Meta smart glasses documenting
ongoing targeting activity. Each entry cross-references the written incident
log (CURRENT_ACTIVITY_LOG.md) and the pattern framework (INVESTIGATIVE_RECORD.md
Section 7G — Four-Element Capture Framework).

---

## METADATA PRESERVATION RULES

- **Never delete original files** — even clips that seem minor
- **Never re-record or screenshot** — always export original file
- **Never run through conversion tools** before attorney review
- **Always note filename exactly** as it appears on device/export
- **Back up immediately:** Glasses → Phone → ProtonDrive + encrypted external drive
- **Do NOT use:** iCloud, Google Photos, Dropbox, or any potentially compromised platform

---

## CONFIDENCE KEY

- ✅ **SOURCED** — clear, unambiguous footage with intact metadata
- 🔶 **PLAUSIBLE** — footage consistent with pattern but requires context
- ❓ **SPECULATIVE** — unclear footage; value depends on surrounding context
- 🔴 **UNCONFIRMED** — footage exists but not yet reviewed/cataloged

## NOTES ON OPERATIONAL SECURITY FOR FOOTAGE

- Do not discuss specific clips in unsecured communications
- Do not show footage to anyone before attorney review
- If glasses are lost or stolen — note date and circumstances immediately;
  report to attorney; this itself becomes evidence
- If you believe footage has been accessed or tampered with —
  note immediately; preserve device without further use
- Keep glasses charged and storage cleared regularly so capture
  is always available
- Consider a secondary backup device (phone camera) for high-activity moments
